# 🎮 Java Quiz Application (Swing GUI)

A modern **Quiz Application built using Java Swing** that provides an interactive GUI-based quiz experience with lifelines and reward system.

## 🚀 Features

- 🖥️ GUI-based application using Java Swing  
- 🎯 Multiple-choice quiz system  
- 💰 Dynamic reward tracking  
- 🎮 50-50 Lifeline  
- ❌ Game ends on wrong answer  
- 🎉 Final winning amount display  

## 🛠️ Technologies Used

- Java  
- Swing (JFrame, JButton, JLabel)  
- AWT (Event Handling)  

## 📌 How It Works

1. User starts the quiz  
2. Questions are displayed one by one  
3. User selects an option using buttons  
4. Correct answer → next question + reward increases  
5. Wrong answer → game ends  
6. Lifeline helps eliminate wrong options
